# D2 JavaScript 2 — 입력 순서를 유지한 중복 제거

- 권장 제한시간: 20분
- 풀이 파일: `d2_problem2.js`
- 목표 시간복잡도: 평균 `O(n)`
- 목표 공간복잡도: `O(k)`
  - `n`: 입력 배열의 전체 원소 수
  - `k`: 서로 다른 정수의 수

## 함수 형태

```javascript
function uniqueInOrder(numbers) {
  // 구현
}
```

## 문제

정수 배열 `numbers`가 주어진다. 중복된 정수를 제거하되, 각 정수가 처음 등장한 순서를 유지한 새 배열을 반환하라.

## 입력 조건

1. 정상 입력은 정수로만 구성된 배열이다.
2. 정수는 음수, 0, 양수일 수 있다.
3. 배열 길이의 최댓값은 `100,000`이다.
4. 같은 정수가 여러 번 등장할 수 있다.

## 반환 규칙

1. 각 정수는 결과 배열에 한 번만 포함한다.
2. 같은 정수가 여러 번 등장하면 최초 등장만 결과에 포함한다.
3. 결과 배열의 순서는 각 정수가 입력에서 처음 등장한 순서와 같아야 한다.
4. 입력이 배열이 아니면 빈 배열 `[]`을 반환한다.
5. 입력이 빈 배열이면 빈 배열 `[]`을 반환한다.
6. 원본 배열을 정렬하거나 변경하지 않는다.
7. 반환값은 원본 배열과 다른 새 배열이어야 한다.

## 예제 1

```javascript
uniqueInOrder([4, 2, 4, 1, 2, 3]);
// [4, 2, 1, 3]
```

설명:

```text
4: 인덱스 0에서 최초 등장
2: 인덱스 1에서 최초 등장
4: 이미 등장했으므로 제외
1: 인덱스 3에서 최초 등장
2: 이미 등장했으므로 제외
3: 인덱스 5에서 최초 등장
```

## 예제 2

```javascript
uniqueInOrder([-1, -1, 0, -1, 2, 0]);
// [-1, 0, 2]
```

## 예제 3

```javascript
uniqueInOrder([7, 7, 7]);
// [7]

uniqueInOrder([5]);
// [5]
```

## 예제 4

```javascript
uniqueInOrder([]);
// []

uniqueInOrder(null);
// []
```

## 추가 테스트

다음 상황을 직접 테스트한다.

- 중복이 없는 배열
- 모든 원소가 같은 배열
- 음수와 0이 포함된 배열
- 원소가 하나인 배열
- 빈 배열
- `null`, 문자열, 객체
- 함수를 호출한 뒤 원본 배열이 그대로인지 확인
- 반환 배열이 원본 배열과 다른 객체인지 확인
