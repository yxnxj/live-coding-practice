# JavaScript 문제 2 — 두 차량 가격의 합

- 제한시간: 20분
- 구현 파일: `problem2.js`

## 함수 형태

```javascript
function findVehiclePair(prices, target) {
  // 구현
}
```

## 요구사항

1. 조건을 만족하는 두 인덱스를 `[작은 인덱스, 큰 인덱스]` 형태로 반환한다.
2. 같은 배열 원소를 두 번 사용할 수 없다.
3. 조건을 만족하는 쌍이 여러 개라면 두 번째 인덱스가 가장 작은 쌍을 반환한다.
4. 두 번째 인덱스도 같다면 첫 번째 인덱스가 가장 작은 쌍을 반환한다.
5. 조건을 만족하는 쌍이 없으면 빈 배열을 반환한다.
6. 입력이 배열이 아니거나 원소가 2개 미만이면 빈 배열을 반환한다.
7. 원본 배열을 변경하지 않는다.
8. 평균 `O(n)` 시간에 해결하는 것을 목표로 한다.

## 예제

```javascript
findVehiclePair([12000, 9000, 15000, 7000], 16000);
// [1, 3]

findVehiclePair([8000, 8000], 16000);
// [0, 1]

findVehiclePair([5000, 6000, 7000], 20000);
// []
```
