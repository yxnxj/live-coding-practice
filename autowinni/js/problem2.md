# JavaScript 문제 2 — 목표 금액을 만드는 두 차량 찾기

- 제한시간: 20분
- 구현 파일: `problem2.js`

## 함수 형태

```javascript
function findVehiclePair(prices, target) {
  // 구현
}
```

## 문제

차량 가격 배열 `prices`와 목표 금액 `target`이 주어진다.

다음 세 조건을 모두 만족하는 두 인덱스 `i`, `j`를 찾아 `[i, j]`로 반환하라.

```text
0 <= i < j < prices.length
prices[i] + prices[j] === target
i와 j는 서로 다른 배열 위치
```

`prices`의 각 원소는 양의 정수이며 같은 가격이 여러 번 등장할 수 있다. `target`도 양의 정수라고 가정한다.

## 요구사항

1. `prices[i] + prices[j] === target`인 두 인덱스를 반환한다.
2. 결과는 항상 `i < j`인 `[i, j]` 형태여야 한다.
3. 하나의 배열 원소를 두 번 사용할 수 없다. 단, 같은 가격이 서로 다른 위치에 두 번 존재하면 각각 사용할 수 있다.
4. 가능한 쌍이 여러 개라면 `j`가 가장 작은 쌍을 반환한다.
5. `j`가 같은 가능한 쌍이 여러 개라면 그중 `i`가 가장 작은 쌍을 반환한다.
6. 가능한 쌍이 없으면 빈 배열을 반환한다.
7. 입력이 배열이 아니거나 원소가 2개 미만이면 빈 배열을 반환한다.
8. 원본 배열을 변경하지 않는다.
9. 평균 `O(n)` 시간에 해결하는 것을 목표로 한다.

## 예제

```javascript
findVehiclePair([12000, 9000, 15000, 7000], 16000);
// [1, 3]

findVehiclePair([8000, 8000], 16000);
// [0, 1]

findVehiclePair([5000, 6000, 7000], 20000);
// []

findVehiclePair([4000, 4000, 6000], 10000);
// 가능한 쌍은 [0, 2], [1, 2]
// j가 같으므로 i가 작은 [0, 2] 반환
```
