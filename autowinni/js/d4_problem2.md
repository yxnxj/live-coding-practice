# D4 JavaScript 2 — 길이가 정해진 연속 구간의 최대 합

- 권장 제한시간: 25분
- 풀이 파일: `d4_problem2.js`

## 함수 형태

```javascript
function maxConsecutiveSum(numbers, length) {
  // 구현
}
```

## 문제

정수 배열 `numbers`와 양의 정수 `length`가 주어진다. 정확히 `length`개의 서로 이웃한 원소로 구성된 모든 연속 구간 중 가장 큰 합을 반환하라.

## 입력 조건

정상 입력은 다음 조건을 만족한다.

1. `numbers`는 비어 있지 않은 정수 배열이다.
2. 각 숫자는 `-1,000,000` 이상 `1,000,000` 이하이다.
3. 음수, `0`, 양수가 포함될 수 있다.
4. `numbers`의 길이는 최대 `200,000`이다.
5. `length`는 `1` 이상 `numbers.length` 이하인 정수다.

정상 배열의 개별 원소가 정수인지는 검사하지 않아도 된다.

## 반환 규칙

1. 정확히 `length`개의 연속된 원소로 만들 수 있는 가장 큰 합을 반환한다.
2. `numbers`가 배열이 아니거나 빈 배열이면 `null`을 반환한다.
3. `length`가 정수가 아니거나 `1`보다 작거나 배열 길이보다 크면 `null`을 반환한다.
4. 원본 `numbers`를 변경하지 않는다.

## 예제 1

```javascript
maxConsecutiveSum([2, 1, 5, 1, 3, 2], 3);
```

길이가 3인 연속 구간의 합:

```text
[2, 1, 5] → 8
[1, 5, 1] → 7
[5, 1, 3] → 9
[1, 3, 2] → 6
```

기대 결과:

```javascript
9
```

## 예제 2

```javascript
maxConsecutiveSum([-5, -2, -8, -1], 2);
```

기대 결과:

```javascript
-7
```

모든 값이 음수여도 실제 구간의 합을 반환해야 한다.

## 추가로 확인할 조건

- `length`가 1인 경우
- `length`가 배열 전체 길이와 같은 경우
- 모든 값이 음수인 경우
- 최대 구간이 배열의 처음이나 끝에 있는 경우
- 최대 합이 같은 구간이 여러 개인 경우
- 빈 배열과 배열이 아닌 입력
- `length`가 0, 음수, 소수 또는 배열 길이보다 큰 경우
- 원본 배열 보존
