# D7 JavaScript 2 — 한 번의 거래

- 제한시간: 25분
- 풀이 파일: `d7_problem2.js`

## 함수 형태

```javascript
function bestTrade(prices) {
  // 구현
}
```

## 문제

날짜 순서대로 기록된 차량 가격 배열 `prices`가 주어진다. 한 날짜에 차량 한 대를 매수하고 그보다 나중 날짜에 한 번 매도할 때, 가장 큰 이익을 얻는 거래를 반환하라.

각 배열 인덱스를 날짜로 사용하며, 이익은 `매도가 - 매수가`로 계산한다.

## 반환 형태

```javascript
{
  buyIndex: 3,
  sellIndex: 4,
  profit: 6
}
```

## 선택 규칙

1. `buyIndex`는 반드시 `sellIndex`보다 작아야 한다.
2. 얻을 수 있는 이익이 가장 큰 거래를 선택한다.
3. 최대 이익이 같은 거래가 여러 개면 `sellIndex`가 가장 작은 거래를 선택한다.
4. `sellIndex`도 같으면 `buyIndex`가 가장 작은 거래를 선택한다.
5. 양의 이익을 얻을 수 없다면 거래하지 않는다.

## 입력 조건

정상 입력은 다음 조건을 만족한다.

1. `prices`는 `null`이 아닌 정수 배열이며 비어 있을 수 있다.
2. 각 가격은 `1` 이상 `1,000,000` 이하이다.
3. 배열의 길이는 최대 `200,000`이다.

입력 형식의 유효성은 검사하지 않아도 된다.

## 반환 규칙

1. 양의 이익을 얻을 수 있으면 선택한 거래의 `buyIndex`, `sellIndex`, `profit`을 객체로 반환한다.
2. 양의 이익을 얻을 수 없으면 `null`을 반환한다.
3. 원소가 두 개보다 적으면 `null`을 반환한다.

## 예제 1

```javascript
bestTrade([8, 3, 6, 1, 7, 4]);
```

인덱스 `3`에서 가격 `1`로 매수하고, 인덱스 `4`에서 가격 `7`로 매도하면 가장 큰 이익 `6`을 얻는다.

기대 결과:

```javascript
{
  buyIndex: 3,
  sellIndex: 4,
  profit: 6
}
```

## 예제 2

```javascript
bestTrade([9, 7, 5, 3]);
```

어느 날짜에 매수해도 이후에 더 높은 가격으로 매도할 수 없으므로 기대 결과는 다음과 같다.

```javascript
null
```

## 추가로 확인할 조건

- 가격이 계속 오르는 경우
- 가격이 계속 내리는 경우
- 같은 최저 가격이 여러 번 등장하는 경우
- 최대 이익이 같은 거래가 여러 개인 경우
- 가격이 모두 같은 경우
- 원소가 두 개, 한 개 또는 없는 경우
