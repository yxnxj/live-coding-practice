# D8 JavaScript 1 — 겹치는 이용 시간 합치기

- 제한시간: 20분
- 풀이 파일: `d8_problem1.js`

## 함수 형태

```javascript
function mergeIntervals(intervals) {
  // 구현
}
```

## 문제

차량 이용 시간을 나타내는 여러 구간이 배열 `intervals`로 주어진다. 서로 겹치거나 끝점이 맞닿아 있는 모든 구간을 합친 뒤, 시작 시간이 빠른 순서로 반환하라.

각 구간은 다음과 같이 시작 시간과 종료 시간을 담는다.

```javascript
[start, end]
```

## 구간을 합치는 규칙

1. 두 구간이 겹치면 하나의 구간으로 합친다.
2. 한 구간의 종료 시간과 다음 구간의 시작 시간이 같아도 합친다.
3. 하나의 구간이 다른 구간에 완전히 포함돼도 하나로 합친다.
4. 여러 구간이 연속해서 이어지면 모두 하나로 합친다.

예를 들어 `[1, 4]`와 `[4, 7]`은 `[1, 7]`이 된다.

## 입력 조건

1. `intervals`는 정수 구간을 담은 배열이며 비어 있을 수 있다.
2. 각 구간은 정확히 두 정수 `[start, end]`로 구성된다.
3. 모든 구간은 `start <= end`를 만족한다.
4. 시작 시간과 종료 시간은 `-1,000,000` 이상 `1,000,000` 이하이다.
5. 구간의 개수는 최대 `200,000`개다.
6. 입력 구간은 정렬되어 있지 않을 수 있다.
7. 잘못된 입력 형식은 고려하지 않아도 된다.

## 반환 규칙

1. 합쳐진 구간들을 시작 시간이 빠른 순서로 담은 새 배열을 반환한다.
2. 결과의 구간들은 서로 겹치거나 맞닿아 있지 않아야 한다.
3. 빈 배열이 주어지면 빈 배열 `[]`을 반환한다.
4. 원본 `intervals`와 내부 구간 배열의 값과 순서를 변경하지 않는다.

## 예제

```javascript
mergeIntervals([
  [8, 10],
  [1, 3],
  [2, 6],
  [15, 18],
  [10, 12]
]);
```

기대 결과:

```javascript
[
  [1, 6],
  [8, 12],
  [15, 18]
]
```

## 추가로 확인할 조건

- 모든 구간이 떨어져 있는 경우
- 모든 구간이 하나로 합쳐지는 경우
- 끝점만 맞닿은 경우
- 한 구간이 다른 구간에 포함된 경우
- 시작 시간이 같은 구간이 여러 개인 경우
- 음수와 길이가 `0`인 구간이 포함된 경우
- 구간이 하나뿐이거나 없는 경우
- 원본 배열 보존
