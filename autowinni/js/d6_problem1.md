# D6 JavaScript 1 — 반복 없는 가장 긴 연속 문자열

- 제한시간: 25분
- 풀이 파일: `d6_problem1.js`

## 함수 형태

```javascript
function longestUniqueLength(text) {
  // 구현
}
```

## 문제

소문자 영문으로 구성된 문자열 `text`가 주어진다. 같은 문자가 두 번 이상 포함되지 않는 연속 부분 문자열 중 가장 긴 길이를 반환하라.

부분 문자열은 원래 문자열에서 서로 이웃한 문자들로 구성되어야 한다.

## 입력 조건

1. `text`는 소문자 영문 `a`부터 `z`로만 구성된다.
2. 빈 문자열이 주어질 수 있다.
3. 문자열 길이는 최대 `200,000`이다.

문자열이 아닌 입력은 고려하지 않아도 된다.

## 반환 규칙

1. 조건을 만족하는 가장 긴 연속 부분 문자열의 길이를 반환한다.
2. 빈 문자열이면 `0`을 반환한다.
3. 원본 문자열은 변경하지 않는다.

## 예제 1

```javascript
longestUniqueLength("abcabcbb");
```

`"abc"`는 같은 문자가 반복되지 않으며 길이는 3이다.

기대 결과:

```javascript
3
```

## 예제 2

```javascript
longestUniqueLength("abcaefbg");
```

조건을 만족하는 가장 긴 부분 문자열 중 하나는 `"caefbg"`다.

기대 결과:

```javascript
6
```

## 추가로 확인할 조건

- 빈 문자열
- 문자 하나
- 모든 문자가 같은 경우
- 모든 문자가 서로 다른 경우
- 반복 문자가 연속해서 등장하는 경우
- 앞에서 등장한 문자가 멀리 떨어져 다시 등장하는 경우
