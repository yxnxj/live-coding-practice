\# D7 JavaScript 1 — 창고 로봇의 이동

- 제한시간: 30분
- 풀이 파일: `d7_problem1.js`

## 함수 형태

```javascript
function minimumMoves(warehouse) {
  // 구현
}
```

## 문제

직사각형 창고의 지도가 같은 길이의 문자열들로 구성된 배열 `warehouse`로 주어진다. 로봇이 출발 위치에서 도착 위치까지 이동하는 데 필요한 최소 이동 횟수를 반환하라.

## 지도 기호

```text
S: 출발 위치
E: 도착 위치
.: 이동할 수 있는 빈 칸
#: 이동할 수 없는 칸
```

## 이동 규칙

1. 로봇은 한 번에 위, 아래, 왼쪽, 오른쪽 중 한 방향으로 한 칸 이동한다.
2. 지도 밖으로 이동할 수 없다.
3. `#`으로 표시된 칸으로 이동할 수 없다.
4. `S`와 `E`가 있는 칸은 이동할 수 있다.

## 입력 조건

정상 입력은 다음 조건을 만족한다.

1. `warehouse`는 `null`이 아닌 문자열 배열이다.
2. 행의 수와 열의 수는 각각 `1` 이상 `100` 이하이다.
3. 모든 문자열의 길이는 같다.
4. 지도에는 `S`와 `E`가 정확히 하나씩 존재한다.
5. 지도는 `S`, `E`, `.`, `#`으로만 구성된다.

입력 형식의 유효성은 검사하지 않아도 된다.

## 반환 규칙

1. `S`에서 `E`까지 도착하는 데 필요한 최소 이동 횟수를 반환한다.
2. 도착할 수 없으면 `-1`을 반환한다.

## 예제 1

```javascript
minimumMoves([
  "S..#",
  ".#..",
  "...E"
]);
```

기대 결과:

```javascript
5
```

## 예제 2

```javascript
minimumMoves([
  "S#.",
  "###",
  "..E"
]);
```

출발 위치와 도착 위치 사이의 모든 길이 막혀 있으므로 기대 결과는 다음과 같다.

```javascript
-1
```

## 추가로 확인할 조건

- 출발 위치와 도착 위치가 이웃한 경우
- 이동 가능한 경로가 여러 개인 경우
- 막힌 칸을 피해 돌아가야 하는 경우
- 도착 위치에 갈 수 없는 경우
- 행이 하나뿐인 지도
- 열이 하나뿐인 지도
