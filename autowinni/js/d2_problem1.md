# D2 JavaScript 1 — 가장 많이 등장한 문자

- 권장 제한시간: 25분
- 풀이 파일: `d2_problem1.js`
- 목표 시간복잡도: 평균 `O(n)`
- 목표 공간복잡도: `O(k)`
  - `n`: 입력 문자열 길이
  - `k`: 서로 다른 문자 수

## 함수 형태

```javascript
function mostFrequentChar(text) {
  // 구현
}
```

## 문제

영문자와 일반 공백으로 구성된 문자열 `text`가 주어진다. 공백을 제외하고 가장 많이 등장한 영문자 하나를 반환하라.

## 입력 조건

1. 정상 입력에는 영문 대문자 `A-Z`, 영문 소문자 `a-z`, 일반 공백 문자 ` `만 들어온다.
2. 문자열 길이의 최댓값은 `100,000`이다.
3. 같은 영문자의 대문자와 소문자는 같은 문자로 취급한다.
   - `A`와 `a`는 모두 `a`로 집계한다.

## 반환 규칙

1. 결과는 항상 소문자로 반환한다.
2. 일반 공백 ` `은 빈도 계산에서 제외한다.
3. 가장 높은 빈도를 가진 문자가 하나라면 그 문자를 반환한다.
4. 가장 높은 빈도를 가진 문자가 여러 개라면 알파벳 순으로 가장 앞선 문자를 반환한다.
5. `text`가 문자열이 아니면 빈 문자열 `""`을 반환한다.
6. `text`가 빈 문자열이거나 공백만 포함하면 빈 문자열 `""`을 반환한다.
7. 원본 문자열을 변경하지 않는다.

## 예제 1

```javascript
mostFrequentChar("Banana Apple");
// "a"
```

설명:

```text
a: 4회
n: 2회
p: 2회
b, l, e: 각 1회
```

따라서 `a`를 반환한다.

## 예제 2

```javascript
mostFrequentChar("bbaacc");
// "a"
```

설명:

```text
a: 2회
b: 2회
c: 2회
```

빈도가 같으므로 알파벳 순으로 가장 앞선 `a`를 반환한다.

## 예제 3

```javascript
mostFrequentChar("zz YY");
// "y"
```

`z`와 `y`가 각각 2회 등장하고, 알파벳 순으로 `y`가 앞선다.

## 예제 4

```javascript
mostFrequentChar("   ");
// ""

mostFrequentChar(null);
// ""
```

## 추가 테스트

다음 상황을 직접 테스트한다.

- 문자 하나만 있는 문자열
- 동일 문자가 대문자와 소문자로 섞인 문자열
- 가장 높은 빈도의 문자가 여러 개인 문자열
- 문자의 앞뒤와 중간에 공백이 있는 문자열
- 빈 문자열
- 공백만 있는 문자열
- `null`, 배열, 숫자
