## 함수 형태

```javascript
function sortVehicles(vehicles) {
  // 구현
}
```

## 문제

차량 객체 배열 `vehicles`가 주어진다. 원본 배열을 변경하지 않고 정해진 우선순위에 따라 정렬한 새 배열을 반환하라.

## 차량 모델

```javascript
{
  id: "V-100",
  price: 15000,
  year: 2023
}
```

정상 입력 배열의 모든 차량은 다음 조건을 만족한다고 가정한다.

1. `id`는 비어 있지 않은 문자열이다.
2. `price`는 0보다 큰 유한한 숫자다.
3. `year`는 정수다.

개별 차량 객체의 유효성 검사는 구현하지 않아도 된다.

## 정렬 규칙

다음 우선순위로 정렬한다.

1. `price` 내림차순
2. 가격이 같으면 `year` 내림차순
3. 가격과 연식이 모두 같으면 `id` 오름차순

## 반환 규칙

1. 정렬된 새 배열을 반환한다.
2. 원본 `vehicles` 배열의 원소 순서를 변경하지 않는다.
3. 반환 배열은 원본 배열과 다른 배열 객체여야 한다.
4. 차량 객체 자체를 복사할 필요는 없다.
5. `vehicles`가 배열이 아니면 빈 배열 `[]`을 반환한다.
6. 빈 배열이면 새로운 빈 배열 `[]`을 반환한다.

## 예제 입력

```javascript
const vehicles = [
  { id: "V-103", price: 15000, year: 2022 },
  { id: "V-101", price: 15000, year: 2023 },
  { id: "V-200", price: 12000, year: 2025 },
  { id: "V-100", price: 15000, year: 2023 },
  { id: "V-050", price: 18000, year: 2020 }
];
```

## 기대 결과

```javascript
[
  { id: "V-050", price: 18000, year: 2020 },
  { id: "V-100", price: 15000, year: 2023 },
  { id: "V-101", price: 15000, year: 2023 },
  { id: "V-103", price: 15000, year: 2022 },
  { id: "V-200", price: 12000, year: 2025 }
]
```

정렬 이유:

- `V-050`은 가격이 가장 높다.
- `V-100`, `V-101`, `V-103`은 가격이 같다.
- 그중 2023년식이 2022년식보다 먼저 온다.
- `V-100`과 `V-101`은 가격과 연식이 같으므로 ID 오름차순으로 정렬한다.

## 추가 테스트

- 차량 한 대

- 가격만 같은 차량
- 가격과 연식이 모두 같은 차량
- 이미 정렬된 배열
- 역순으로 정렬된 배열
- 빈 배열
- 배열이 아닌 입력
- 함수 호출 후 원본 배열 순서 확인
- 반환 배열과 원본 배열의 참조 비교
