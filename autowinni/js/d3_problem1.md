# D3 JavaScript 1 — 올바른 괄호 문자열

- 권장 제한시간: 25분
- 풀이 파일: `d3_problem1.js`
- 목표 시간복잡도: `O(n)`
- 목표 공간복잡도: `O(n)`
  - `n`: 입력 문자열 길이

## 함수 형태

```javascript
function isValidBrackets(text) {
  // 구현
}
```

## 문제

괄호 문자로 구성된 문자열 `text`가 주어진다. 모든 괄호의 종류와 순서가 올바르게 짝지어져 있는지 판별하라.

## 입력 조건

1. 정상 입력은 다음 여섯 문자로만 구성된다.

```text
( ) { } [ ]
```

2. 문자열 길이의 최댓값은 `100,000`이다.

## 올바른 괄호 문자열

다음 조건을 모두 만족하면 올바른 괄호 문자열이다.

1. 모든 여는 괄호에는 같은 종류의 닫는 괄호가 있다.
2. 가장 최근에 열린 괄호가 가장 먼저 닫힌다.
3. 여는 괄호보다 닫는 괄호가 먼저 등장하지 않는다.
4. 문자열 확인이 끝났을 때 닫히지 않은 여는 괄호가 남아 있지 않는다.

## 반환 규칙

1. 올바른 괄호 문자열이면 `true`를 반환한다.
2. 올바르지 않으면 `false`를 반환한다.
3. 빈 문자열 `""`은 올바른 괄호 문자열로 보고 `true`를 반환한다.
4. `text`가 문자열이 아니면 `false`를 반환한다.

## 예제

```javascript
isValidBrackets("()[]{}");
// true

isValidBrackets("([{}])");
// true

isValidBrackets("([)]");
// false

isValidBrackets("((");
// false

isValidBrackets("]");
// false

isValidBrackets("");
// true

isValidBrackets(null);
// false
```

## 추가 테스트

- 괄호 한 쌍
- 여러 종류가 나란히 있는 문자열
- 여러 종류가 중첩된 문자열
- 닫는 괄호가 먼저 등장하는 문자열
- 괄호 종류가 맞지 않는 문자열
- 여는 괄호가 남는 문자열
- 빈 문자열
- `null`, 배열, 숫자
