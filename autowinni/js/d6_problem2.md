# D6 JavaScript 2 — 정사각 행렬 회전

- 제한시간: 25분
- 풀이 파일: `d6_problem2.js`

## 함수 형태

```javascript
function rotateClockwise(matrix) {
  // 구현
}
```

## 문제

`n × n` 정사각 행렬 `matrix`가 주어진다. 모든 값을 시계 방향으로 90도 회전한 새 행렬을 반환하라.

## 입력 조건

정상 입력은 다음 조건을 만족한다.

1. `matrix`는 배열이다.
2. 각 원소는 길이가 `n`인 배열이다.
3. 모든 값은 정수다.
4. `n`은 `0` 이상 `500` 이하이다.

개별 행과 값의 유효성은 검사하지 않아도 된다.

## 반환 규칙

1. 시계 방향으로 90도 회전한 새 행렬을 반환한다.
2. 빈 행렬 `[]`이면 빈 배열 `[]`을 반환한다.
3. 원본 행렬과 원본 행의 값과 순서를 변경하지 않는다.
4. 반환된 바깥 배열과 각 행은 모두 원본과 다른 배열 객체여야 한다.

## 예제

입력:

```javascript
[
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]
```

기대 결과:

```javascript
[
  [7, 4, 1],
  [8, 5, 2],
  [9, 6, 3]
]
```

## 추가로 확인할 조건

- `1 × 1` 행렬
- `2 × 2` 행렬
- 홀수 크기와 짝수 크기
- 음수와 `0`이 포함된 행렬
- 빈 행렬
- 원본 행렬과 각 행 보존
