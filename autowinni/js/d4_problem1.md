# D4 JavaScript 1 — 정렬된 두 배열의 공통 값

- 풀이 파일: `d4_problem1.js`

## 함수 형태

```javascript
function commonSortedNumbers(first, second) {
  // 구현
}
```

## 문제

오름차순으로 정렬된 두 정수 배열 `first`와 `second`가 주어진다. 두 배열에 모두 등장하는 숫자들을 오름차순으로 담은 새 배열을 반환하라.

## 입력 조건

정상 입력은 다음 조건을 만족한다.

1. `first`와 `second`는 정수 배열이다.
2. 두 배열은 이미 오름차순으로 정렬되어 있다.
3. 음수, `0`, 양수와 중복값이 포함될 수 있다.
4. 각 배열의 길이는 최대 `100,000`이다.

정상 배열의 개별 원소가 정수인지와 정렬되어 있는지는 검사하지 않아도 된다.

## 중복 처리 규칙

같은 숫자가 두 배열에 여러 번 등장하면, 두 배열에서 등장한 횟수 중 작은 횟수만큼 결과에 포함한다.

```javascript
first  = [2, 2, 2];
second = [2, 2];

// 결과
[2, 2]
```

## 반환 규칙

1. 두 배열에 모두 등장하는 숫자들을 오름차순으로 담은 새 배열을 반환한다.
2. 둘 중 하나라도 배열이 아니면 빈 배열 `[]`을 반환한다.
3. 둘 중 하나라도 빈 배열이면 빈 배열 `[]`을 반환한다.
4. 원본 배열의 값과 순서를 변경하지 않는다.

## 예제 1

```javascript
const first = [1, 2, 2, 4, 7];
const second = [2, 2, 3, 7, 9];
```

기대 결과:

```javascript
[2, 2, 7]
```

## 예제 2

```javascript
const first = [-3, -1, 0, 4];
const second = [-2, -1, 0, 0, 5];
```

기대 결과:

```javascript
[-1, 0]
```

## 추가로 확인할 조건

- 공통 값이 없는 경우
- 모든 값이 공통인 경우
- 같은 숫자의 중복 횟수가 다른 경우
- 음수와 `0`이 포함된 경우
- 한 배열의 길이가 1인 경우
- 빈 배열과 배열이 아닌 입력
- 원본 배열 보존
