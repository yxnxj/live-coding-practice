# D5 Java — 연속된 문자 구간 요약

- 권장 제한시간: 25분
- 풀이 파일: `D5Solution.java`

## 메서드 형태

```java
public static String summarizeRuns(String text) {
    // 구현
}
```

## 문제

소문자 영문으로 구성된 문자열 `text`가 주어진다. 왼쪽부터 확인하면서 같은 문자가 연속된 각 구간을 `문자 + 연속 횟수` 형태로 바꿔 하나의 문자열로 반환하라.

## 변환 규칙

1. 문자열의 왼쪽부터 오른쪽 순서로 처리한다.
2. 같은 문자가 연속되는 동안 하나의 구간으로 본다.
3. 각 구간을 해당 문자와 구간 길이를 이어 붙인 형태로 변환한다.
4. 길이가 1인 구간도 숫자 `1`을 붙인다.
5. 구간 길이가 10 이상이면 전체 숫자를 그대로 붙인다.

## 입력 조건

정상 입력은 다음 조건을 만족한다.

1. `text`는 소문자 영문 `a`부터 `z`로만 구성된다.
2. `text`는 `null`이 아니며 빈 문자열일 수 있다.
3. 문자열 길이는 최대 `200,000`이다.

개별 문자가 소문자 영문인지 검사하지 않아도 된다.

## 반환 규칙

1. 규칙에 따라 변환한 문자열을 반환한다.
2. `text`가 빈 문자열이면 빈 문자열 `""`을 반환한다.

## 예제 1

```java
summarizeRuns("aaabbccccdaa");
```

구간:

```text
aaa | bb | cccc | d | aa
```

기대 결과:

```java
"a3b2c4d1a2"
```

## 예제 2

```java
summarizeRuns("abab");
```

기대 결과:

```java
"a1b1a1b1"
```

## 추가로 확인할 조건

- 문자가 하나뿐인 경우
- 모든 문자가 같은 경우
- 모든 문자가 서로 다른 경우
- 처음과 마지막 문자가 같지만 중간에 다른 문자가 있는 경우
- 한 구간의 길이가 10 이상인 경우
- 빈 문자열
