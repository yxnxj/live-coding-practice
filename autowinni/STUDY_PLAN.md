## 연습 원칙

1. VS Code만 사용한다.
2. 검색, AI, 기존 풀이를 보지 않고 먼저 푼다.
3. Java 1문제와 JavaScript 2문제를 한 세트로 구성한다.
4. 문제 파일에는 사용할 알고리즘, 자료구조, 목표 시간복잡도를 미리 적지 않는다.
5. 문제의 조건과 입력 크기를 보고 해결 방법을 스스로 선택한다.
6. 첫 풀이에서는 가장 단순한 방법이라도 완성한다.
7. 풀이 후 선택한 방법과 시간복잡도, 공간복잡도를 기록한다.
8. 같은 패턴의 연습은 최대 두 번까지만 하고 이후에는 다른 유형을 우선한다.
9. 정상 입력 외에 빈 입력, 원소 1개, 중복값, 동일 정렬값을 테스트한다.

## 파일명 규칙

```text
Java 문제 설명: src/d{일차}_problem.md
Java 풀이:      src/D{일차}Solution.java

JS 문제 설명:   js/d{일차}_problem1.md
JS 풀이:        js/d{일차}_problem1.js
JS 문제 설명:   js/d{일차}_problem2.md
JS 풀이:        js/d{일차}_problem2.js
```

오늘 만든 기존 파일은 이름을 변경하지 않는다.
