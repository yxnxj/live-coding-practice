1. VS Code만 사용한다.
2. 검색, AI, 기존 풀이를 보지 않고 먼저 푼다.
3. Java 1문제와 JavaScript 2문제를 한 세트로 구성한다.
4. 첫 풀이에서는 가장 단순한 방법이라도 완성한다.
5. 풀이 후 시간복잡도와 공간복잡도를 기록한다.
6. 정상 입력 외에 빈 입력, 원소 1개, 중복값, 동일 정렬값을 테스트한다.
