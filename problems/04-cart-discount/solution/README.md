# Cart Discount Kata

장바구니 금액, 할인 정책, 배송비를 계산하는 도메인 로직 연습이다.

## 문제 요약

이 문제는 원본 카타가 아니라 라이브 코딩 대비용 커스텀 문제다.

- 상품에는 이름, 카테고리, 단가, 수량이 있다.
- 쿠폰 할인, 카테고리 할인, 무료배송 조건을 계산한다.
- 할인 적용 순서와 할인 내역을 명확히 반환한다.

## 테스트 실행

```bash
./gradlew runKataTests
```

IntelliJ IDEA에서는 이 `solution/` 폴더를 Gradle 프로젝트로 연다.
