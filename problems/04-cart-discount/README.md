# 04 Cart Discount

## Type

Domain logic

## Source

- Custom practice problem
- 유명 카타 원본이 아니라 라이브 코딩에서 자주 나오는 도메인 규칙 구현형 문제로 만든 연습 문제다.

## Practice Goal

- 장바구니 금액, 쿠폰, 카테고리 할인, 무료배송 조건을 계산한다.
- 할인 적용 순서를 명확히 표현한다.
- 최종 금액과 할인 내역을 함께 반환한다.

## Suggested Requirements

- 상품에는 이름, 카테고리, 단가, 수량이 있다.
- 쿠폰 할인과 카테고리 할인은 정해진 우선순위로 적용된다.
- 무료배송 조건을 만족하면 배송비가 0원이 된다.
- 결과에는 원금, 할인 목록, 배송비, 최종 금액이 포함된다.

## Done Criteria

- 할인 우선순위 테스트가 있다.
- 새 할인 정책 추가 시 바꿀 위치가 명확하다.
