# 08 Mars Rover

## Type

State / Command handling

## Practice Goal

- 위치와 방향 상태를 명령에 따라 변경한다.
- 회전, 이동, 명령 파싱 책임을 분리한다.
- 장애물 같은 요구사항을 점진적으로 추가한다.

## Source

- Common Mars Rover Kata variant
- 이 저장소에서는 특정 단일 원본에 맞추기보다, 라이브 코딩에서 자주 쓰이는 대표 요구사항으로 연습한다.

## 대표 문제 요약

화성 탐사 로버가 격자 위에서 현재 위치와 방향을 가진다.

- 위치는 보통 `(x, y)` 좌표로 표현한다.
- 방향은 `N`, `E`, `S`, `W` 중 하나다.
- `L` 명령은 왼쪽으로 90도 회전한다.
- `R` 명령은 오른쪽으로 90도 회전한다.
- `M` 명령은 현재 방향으로 한 칸 이동한다.
- 여러 명령을 문자열로 받아 순서대로 실행한다.
- 확장 요구로 격자 경계, wrapping, 장애물 감지 등을 추가할 수 있다.

## Done Criteria

- `L`, `R`, `M` 명령 테스트가 있다.
- 방향 전환 규칙이 명확하게 표현되어 있다.
- 장애물 처리 확장 지점이 보인다.
